# 🎹 لغة البرمجة الموسيقية الرسمية - Music Language

أهلاً بكم في لغتي الموسيقية المبتكرة المصممة بالكامل بلغة C! 🚀

## ✨ المميزات:
- سهلة جداً ولا تستخدم التابات المعقدة.
- تعتمد على مسافة واحدة فقط بين النغمات لتنظيم المقطوعة.
- واجهة رسومية كاملة (C-GUI Compiler).

## 🎼 كلمات اللغة السحرية:
- **Do, Re, Mi, Fa, Sol, La, Si**: النغمات الموسيقية الأساسية.
- **bosishen**: للتحكم في حركة النغمات (طلوع ونزول).
- **NewLine**: لعمل سطر جديد من فوق وتنسيق المقطوعة.
- **underLine**: لعمل سطر جديد من تحت للفصل بين العزف.

## 🚀 طريقة التشغيل:
حمل ملف `music_app.zip` من قسم الإصدارات (Releases)، فك الضغط، واضغط على زر **Compile _Play Piano** واستمتع بالعزف!
